Question-1: What are the different ways to select an element in the DOM?

The different ways to select elements in the DOM using methods like getElementById , getElementsByClassName , getElementsByTagName , querySelector, and querySelectorAll .

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML returns or sets HTML content, including tags. innerText returns or sets only visible text, ignoring hidden elements. textContent returns or sets all text content, including hidden elements, but ignores HTML tags

Question-3: What is event delegation in the DOM?

Event Delegation is a technique where you attach a single event listener to a parent element to handle events for all of its child elements.

Question-4: What is event bubbling in the DOM?

Event Bubbling is a behavior in the DOM where an event triggered on a child element propagates up through its ancestors in the DOM tree.

Question-5: How do you create, add, and remove elements using JavaScript?

To create an element, use document.createElement(). To add it, use appendChild() or insertBefore(). To remove an element, use removeChild() or the remove() method. You can also modify elements by setting properties like textContent or innerHTML.
